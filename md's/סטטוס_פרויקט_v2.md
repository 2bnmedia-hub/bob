## טכנולוגיה
- Next.js 16.2.9 + TypeScript + Tailwind CSS v4
- גופן: Heebo — נטען דרך `<link>` ב-HTML (לא next/font) כדי למנוע חסימת startup
- `next.config.mjs` (לא `.ts` — SWC לא תומך ב-.ts על darwin/arm64)
- Supabase לבסיס נתונים
- Vercel לדיפלוי

## בעיות ידועות שנפתרו
1. **SWC binary חסר** (darwin/arm64) → נפתר עם `npm install --force`
2. **next.config.ts** גורם לשגיאה → שונה ל-`next.config.mjs`
3. **גופן Heebo** חוסם startup → הועבר ל-`<link>` ב-layout.tsx
4. **Turbopack קומפילציה ראשונה** — איטית (כמה דקות), בסבלנות

## הרצת dev server

## מבנה הפרויקט
```
src/
  app/
    page.tsx          ← דף הבית
    layout.tsx        ← RTL, Heebo font, CartProvider
    globals.css       ← צבעי מותג, base styles
    dashboard/page.tsx ← לוח בקרה (CRUD + import/export)
    cart/page.tsx     ← סל קניות
    about/page.tsx
    contact/page.tsx
    category/[slug]/page.tsx

    terms, privacy, accessibility, faq, account/
  components/
    Header.tsx        ← Navbar משופר (3 שכבות)
    GallerySlider.tsx ← סליידר fullscreen אוטומטי
    DepartmentBanners.tsx ← bento grid
    PromoBanners.tsx
    FeaturedProducts.tsx ← כרטיסי מוצר עם כפתור סל
    ArticleCards.tsx
    TrustSection.tsx
    Newsletter.tsx
    Footer.tsx
    PlaceholderPage.tsx
  context/
    CartContext.tsx   ← state גלובלי לסל
  lib/
    supabase.ts       ← Supabase client
    importExport.ts   ← CSV/Excel/JSON import/export
```
